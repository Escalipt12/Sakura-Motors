# Технологический стек

## Программное обеспечение для проектирования

- **Draw.io** — сервис для создания схем, диаграмм, ER-диаграмм и архитектуры.
- **Figma** — сервис для создания макетов интерфейса, логотипа, иконок.

## Программное обеспечение для разработки

### Фронтэнд

- **TypeScript**
- **React**
- **Mantine**
- **Vite**

### Бэкенд

- **Node.js**
- **Express**
- **TypeScript**
- **Prisma**

### Базы данных

- **PostgreSQL**
- **Redis**
